# Media Miner Frontend

Frontend hosted on Firebase with Authentication support.

This repository contains the frontend of the Media Miner application. Media Miner is designed to allow users to download audio and video content from websites like YouTube, Vimeo, and more. 

## Features
- Firebase Hosting: Deliver your app to users globally.
- Firebase Authentication: Secure login and user management.

## Languages Used
- JavaScript
- CSS
- HTML

---
## Getting Started

### Prerequisites
- [Firebase CLI](https://firebase.google.com/docs/cli/)
- [Node.js](https://nodejs.org/)

### Installation
1. Clone the repository:
```bash
git clone https://github.com/DaemonCypher/MediaMinerFrontend.git
```
2. Navigate to the project folder:
```bash
cd MediaMinerFrontend
```
3. Install the dependencies:
```bash
npm install
```

### Deploying
1. Log in to Firebase:
```bash
firebase login
```
2. Deploy using Firebase CLI:
```bash
firebase deploy
```
